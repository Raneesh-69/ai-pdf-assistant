# 📄 AI PDF Assistant

An AI-powered PDF analysis assistant built using Streamlit, Groq LLM, and PyPDF.

## Users can upload PDF documents, generate summaries, extract key insights, and ask questions about the document using natural language.

🚀 Live Demo
AI PDF Assistant:https://ai-pdf-assistant-kk76ju58bbxhva4qktpvcz.streamlit.app/

---

## Features

- 📄 Upload PDF documents
- 🤖 AI-powered question answering
- 📋 Automatic document summarization
- 💬 Chat interface with conversation history
- ⚡ Fast responses using Groq LLM
- 🌐 Deployable on Streamlit Cloud

## Tech Stack

- Python
- Streamlit
- Groq API
- Llama 3.3 70B
- PyPDF

## Usage

1. Upload a PDF document.
2. Generate a document summary.
3. Ask questions about the uploaded document.
4. Receive AI-generated answers based on the document content.
